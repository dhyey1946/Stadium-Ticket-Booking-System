# Stadium Ticket Booking Management System

A Java Console-Based Stadium Ticket Booking System developed using Java, JDBC, and MySQL.

## Features

### User Module
- User Registration
- User Login
- Change Password
- Delete Account
- View Matches
- Book Tickets
- Seat Selection
- SMS Ticket Simulation

### Staff Module
- Staff Login
- Add Match
- Remove Match
- View User Details
- View Match Details
- Revenue Reports
- Generate Bills

### Technologies Used
- Java
- JDBC
- MySQL
- OOP Concepts
- Collections Framework
- File Handling
